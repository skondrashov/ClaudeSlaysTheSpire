# Decision Boundaries

The attention graph's top level. Each file is the index for one recurring decision moment: what to consider when you are there. Entries have one shape — *consider X* — where X is an entry (linked), an aspect of the live state, or a question. The reasoning is always yours; these files only put the right things in front of it.

- Content lives where it lives; these indexes point. A consideration with no home page is stated inline.
- You are aware of the current game state by default — no entry exists to say "look at your HP." Entries earn their place by naming what is NOT salient: configurations, futures, and connections.
- Awareness is either *noticing* (configurations the state shows but you might not weigh — the [[domain:sts1, category:recognitions, potion-use|recognitions]]) or *preparing* (what is about to matter — these boundary files).
