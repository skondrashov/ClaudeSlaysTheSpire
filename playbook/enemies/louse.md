# Louse (hallway, Act 1, HP: ~11-17, appears in groups of 2-3)
PATTERN: Attacks for ~5-7 damage per turn. Has Curl Up passive (gains 5-7 block when first attacked).
KEY MECHANIC: Curl Up absorbs first hit. In groups, combined damage adds up.
WHAT THIS MEANS: Focus fire one at a time. The first attack triggers Curl Up block, so follow up with more attacks that turn.
BENCHMARKS: Kill one per turn with Vulnerable.
BRING: AOE if facing 3. Focus fire.
AVOID: Splitting damage across all Louses.
