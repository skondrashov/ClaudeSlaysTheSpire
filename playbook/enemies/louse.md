# Louse (hallway, Act 1, appears in groups of 2)

Two types: Red Louse and Green Louse (the game calls Green Louse what the wiki calls "Green Louse" -- it applies Weak).

## Red Louse — HP: 10-15
- Bite: Deals N damage (N = random 5-7, set at combat start). 75% chance per turn.
- Grow: Gains 3 Strength. 25% chance per turn.
- Cannot use same move three times in a row.
- Curl Up: Gains 3-7 Block when first hit by an attack.

## Green Louse — HP: 11-17
- Bite: Deals N damage (N = random 5-7, set at combat start). 75% chance per turn.
- Spit Web: Applies 2 Weak. 25% chance per turn.
- Cannot use same move three times in a row.
- Curl Up: Gains 3-7 Block when first hit by an attack.

KEY MECHANIC: Curl Up absorbs first hit. In groups, combined damage adds up. Red Louse's Grow (+3 Str) makes it scale if left alive.
WHAT THIS MEANS: Focus fire one at a time. The first attack triggers Curl Up block, so follow up with more attacks that turn. Kill Red Louse first if it uses Grow (Strength scaling is more dangerous than Weak application).
BENCHMARKS: Kill one per turn with Vulnerable.
BRING: AOE if facing 3. Focus fire.
AVOID: Splitting damage across all Louses.

DANGER LEVEL: Low. Standard Act 1 encounter.
