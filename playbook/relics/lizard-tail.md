# Lizard Tail
EFFECT: When HP reaches 0, heal to 50% Max HP. One-time use, then becomes inactive.
HOW IT CHANGES DECISIONS: A safety net similar to Fairy in a Bottle but heals significantly more (50% Max HP vs ~30%). Save it for boss fights or dangerous elites. Unlike Fairy, Lizard Tail is a relic -- it doesn't take a potion slot. Once triggered, it's gone for the rest of the run.
