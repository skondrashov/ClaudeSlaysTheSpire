# Toy Ornithopter
EFFECT: Whenever you use a potion, heal 5 HP.
HOW IT CHANGES DECISIONS: Turns potion usage into healing. Each potion heals an extra 5 HP. With 3 potion uses in a run: +15 HP total. Minor but free. Consider using potions slightly more aggressively when this relic is active.

ANTI-SYNERGY: Completely dead with Sozu (which prevents obtaining potions). Do NOT take Toy Ornithopter if you have Sozu. If offered from an event and Sozu is already held, skip it.
