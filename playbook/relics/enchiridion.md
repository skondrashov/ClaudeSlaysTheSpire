# Enchiridion
EFFECT: At the start of each combat, add a random Power card to your hand. It costs 0 this turn.
HOW IT CHANGES DECISIONS: Free Power every fight. Can generate Metallicize, Inflame, Corruption, Dark Embrace, or other Powers. The 0 cost means it never competes with other turn 1 plays. High variance -- sometimes generates Berserk (dangerous) or a duplicate of a Power you already have. Overall excellent value for free.
