# Data Disk (Common Relic, Defect-only)

EFFECT: Start each combat with 1 Focus.

WHY IT'S STRONG: Focus is the Defect's primary scaling stat. Each point of Focus adds 1 to every orb passive (Frost block, Lightning damage, Dark scaling). Data Disk provides +1 Focus for free, permanently, from Turn 1 of every combat. With 3 Frost orbs, this is +3 block per turn passively. Over a 10-turn fight, that is 30+ block from a single relic.

KEY INTERACTIONS:
- **Frost orbs:** +1 passive block per Frost orb per turn. With 5 Frost orbs, Data Disk adds 5 passive block per turn.
- **Defragment:** Stacks with Defragment for total Focus. Data Disk + Defragment = Focus 2 at minimum, making orb passives immediately impactful.
- **Loop:** Loop triggers a passive, so Data Disk's Focus bonus applies to Loop's extra passive trigger as well.
- **Emotion Chip:** Same -- Focus applies to all passive triggers, including Emotion Chip's bonus start-of-turn passives.

CONFIRMED: Present in the first Defect victory. Data Disk provided the baseline Focus that made Frost orb passives meaningful from Turn 1, supplementing the orb engine before additional Focus scaling was established.
