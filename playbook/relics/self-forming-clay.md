# Self-Forming Clay (Uncommon Relic)
Whenever you lose HP in combat, gain 3 Block next turn.

EVALUATION: Moderate defensive relic. Provides 3 free block the turn after taking damage. Works well with the Full Block Algorithm -- if zero damage is the goal, this relic only triggers when full block fails, providing a small safety net for the next turn. Does not stack with itself (taking damage multiple turns in a row gives 3 block each subsequent turn, not cumulative).

WHEN IT'S GOOD: Multi-turn fights where occasional chip damage is unavoidable. Provides incremental block that helps reach full block thresholds on subsequent turns (e.g., 3 block from Clay + 2 Defends = 13 block instead of 10).

WHEN IT'S WEAK: Fights where you take zero damage consistently (Clay never triggers) or fights where a single massive hit kills you (3 block next turn is meaningless if dead this turn).
