# Dead Branch (Rare Relic, whenever you exhaust a card, add a random card to your hand)

Every exhausted card is replaced by a random card added to your hand. The random card costs its normal energy but is a free card advantage engine.

WHEN IT'S GOOD: In any deck with exhaust sources. The more cards you exhaust per fight, the more value Dead Branch generates. Transforms exhaust from a resource-spending mechanic into a resource-generating one.

CORRUPTION SYNERGY: The strongest combo in the game. With Corruption active, every Skill costs 0E and exhausts on play, and Dead Branch replaces it with a random card. This means every free Skill played generates a new card. The hand never empties, and you cycle through enormous numbers of cards per turn. Any block or damage cards generated are effectively free because Corruption freed up the energy that would have been spent on Skills. This engine wins any fight it survives to activate.

SETUP RISK: The Corruption + Dead Branch engine requires playing Corruption (3E unupgraded) and having Dead Branch already equipped. On the setup turn, you spend 3E and get no immediate block. Against multi-enemy encounters or high-damage enemies, this setup turn can be fatal. The engine is only as good as your ability to survive activating it.

FEEL NO PAIN SYNERGY: With Feel No Pain active, every exhaust triggers both Dead Branch (new card) and Feel No Pain (3 block). This provides defensive sustain during the engine turns.

DECISION POINTS: Dead Branch is always a strong relic pick. When you have Dead Branch, prioritize cards that exhaust (Corruption, Second Wind, True Grit+, Burning Pact, Fiend Fire). Corruption becomes the highest-priority card to acquire when Dead Branch is in your relic pool.

SILENT EXHAUST SYNERGY: Dead Branch transforms the Silent's many exhaust cards into an infinite value engine. Key interactions:
- **Shivs exhaust after play** (they are Ethereal-like 0-cost attacks). Each Shiv played exhausts and triggers Dead Branch, generating a random card. With 4-6 Shivs per turn, Dead Branch generates 4-6 new cards per turn.
- **Storm of Steel** discards hand and generates Shivs equal to cards discarded. Each Shiv then exhausts on play for Dead Branch. Storm of Steel -> 5 Shivs -> 5 Dead Branch procs = massive card generation chain.
- **Terror** (0E, Vuln 99, exhaust) becomes a free Dead Branch proc on top of its devastating debuff.
- **Endless Agony** (0E, exhaust, copies itself when drawn) provides repeatable Dead Branch triggers.
- **Adrenaline** (0E, +2E, draw 2, exhaust) = free energy + draws + Dead Branch proc.
- **Distraction** (1E, exhaust, generates random Skill) = Dead Branch proc + random Skill.
- **Piercing Wail** (exhaust) = Dead Branch proc after the Str reduction.
- **Crippling Cloud** (exhaust) = Dead Branch proc after poison + Weak AOE.

The Silent Dead Branch engine won against the Awakened One (both phases). In Phase 2, the player avoided all Powers and relied entirely on Dead Branch + exhaust cards for card generation. Storm of Steel -> Shiv chains via Dead Branch created explosive turns dealing 45+ damage while generating block through After Image. This is a confirmed winning approach.

DEAD BRANCH + AFTER IMAGE: Each Dead Branch-generated card played triggers After Image for +1 block. With 5-10 card plays per turn from Dead Branch chains, After Image generates 5-10 passive block per turn. This combination provides both offense (random cards) and defense (After Image block) simultaneously.

ANTI-SYNERGIES: None significant. Dead Branch is universally strong. The only risk is relying on random card generation for specific answers (you cannot control what cards you get).
