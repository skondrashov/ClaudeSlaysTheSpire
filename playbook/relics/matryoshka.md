# Matryoshka
EFFECT: The next 2 chests you open contain 2 relics instead of 1.

HOW IT CHANGES DECISIONS: Pure value relic with no downside. Doubles relic rewards from the next two treasure chests. Makes treasure room pathing slightly more attractive -- if choosing between a path with a chest and one without, the chest path gains extra value. No gameplay impact during combat.

NOTES: Has a charge counter (starts at 2, decrements per chest opened). Once both charges are spent, the relic is inert but remains in the relic bar. Does not affect elite relic drops or boss relic choices -- only treasure chests.
