# Dark Embrace (2E, Power, whenever a card is exhausted, draw 1)
Draw engine for exhaust-heavy decks.

WHEN IT'S GOOD: With multiple exhaust sources (True Grit+, status cards, Ethereal cards). Every exhaust replaces itself in hand.
DECISION POINTS: At 2E, it's expensive to set up. Play on free turns. Powers do NOT trigger Gremlin Nob Enrage. WARNING: encourages over-exhausting, which can thin the deck dangerously.
SYNERGIES: True Grit+ (exhaust + draw every turn), Charon's Ashes (exhaust = draw + 3 AOE damage), Fiend Fire (exhaust entire hand = draw many cards, enabling a second wave of plays), status cards (exhausting Slimed/Dazed draws a real card), Disarm/Disarm+ (exhaust on play = free draw).
