# Talk to the Hand (Watcher Uncommon Attack, 2E)

Base: 5 damage. Apply Block Return 2 to target. Exhaust. Upgraded: 7 damage, Block Return 3.

Whenever the affected enemy attacks you, you gain 2 (3 upgraded) block per hit. Against multi-hit enemies, this triggers once per hit.

## Key Interactions

- **Snake Plant (7x3):** Block Return 2 triggers 3 times per attack = 6 free block per Chomp. With upgraded: 9 free block per Chomp. Extremely strong in this matchup.
- **Multi-hit attacks:** The more hits an enemy does, the more value Block Return provides. Excellent against Hexaghost Inferno (6 hits), Book of Stabbing (multi-hit), and any multi-attack enemy.
- **Exhausts:** Only one use per combat. Apply it to the primary threat early.

## Play Order

Play Talk to the Hand early in the fight for maximum Block Return value over multiple attack turns. Since it exhausts, do not save it for later.
