# Feel No Pain (1E, Power, gain 3 block whenever a card is exhausted)

Passive block engine that triggers on every exhaust. Permanent for the rest of combat.

WHEN IT'S GOOD: Any deck with exhaust sources. Converts Slimed/Wound/Dazed/Burn exhaustion into block. Turns status card removal from a tempo loss into a defensive gain. Strongest in fights that generate status cards (Slime Boss, Sentries, Chosen under Hex).

DECISION POINTS: Play on a free turn (enemy buffing/debuffing) for zero opportunity cost. At 1E, worth playing even when only 2-3 exhaust triggers are expected (6-9 block over a fight). Priority increases with more exhaust sources in the deck.

SYNERGIES: True Grit+ (each exhaust = block from True Grit + 3 from FNP), Burning Pact (exhaust + draw + 3 block), Fiend Fire (exhaust entire hand = 3 block per card), Corruption (all Skills exhaust = constant block triggers -- FNP is critical for surviving Corruption's setup turns by generating block from every subsequent Skill play), Corruption + Dead Branch (the triple combo: every Skill exhausts for free, generates 3 block from FNP, and adds a random card from Dead Branch -- this is the strongest defensive engine in the game), Second Wind (mass exhaust = massive FNP block burst).

SLIME BOSS SYNERGY: Slimed cards cost 1E to exhaust. With Feel No Pain active, each Slimed exhausted generates 3 block, partially offsetting the 1E tempo cost. In a typical Slime Boss fight with 5+ Slimed cards, this generates 15+ total block across the fight.

ANTI-SYNERGY: Weak in decks with few exhaust sources. If the deck only exhausts 1-2 cards per fight, 3-6 block total is poor value for a deck slot.
