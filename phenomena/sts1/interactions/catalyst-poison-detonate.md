# Catalyst poison detonation

- **Applies when:** Silent poison deck has [[cards/Catalyst]] and a target already carrying meaningful Poison ([[cards/Bouncing Flask]] and [[cards/Burst]] amplify it).

[[cards/Catalyst]] doubles (Catalyst+ triples) the [[debuffs/Poison]] currently on a target — its value scales with how much poison is already there, so ramp first and detonate late. [[cards/Bouncing Flask]] applies poison in multiple separate bounces (each a distinct application). [[cards/Burst]] + Catalyst plays Catalyst twice for up to 6x poison. The poison-deck finisher.

Related: [[cards/Catalyst]], [[cards/Bouncing Flask]], [[cards/Burst]], [[debuffs/Poison]]
