# Run Log

## Run 0 — Ironclad A0, Victory (Guardian)
DECK: True Grit+, Shrug It Off+, Pommel Strike
CAUSE OF DEATH: N/A — defeated The Guardian
KEY MOMENTS: Block-and-cycle strategy with multi-purpose cards carried the boss fight. True Grit+ (controlled exhaust) was key.
LESSONS: Quality over quantity — cards that do double duty (block+exhaust, block+draw, damage+draw) outperform single-purpose cards.

## Run 1 — Ironclad A0, Death Floor 16 (Slime Boss)
DECK: 4 Strikes, 4 Defends, Bash+, Berserk, Hemokinesis, Clothesline, Shockwave+, Shrug It Off, Ghostly Armor (15 cards)

LESSONS: Need AOE for Slime Boss split. Don't take cards you don't understand. Don't play Berserk in dangerous fights. Deck thinning matters.
