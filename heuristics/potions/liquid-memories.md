# Liquid Memories

Use when a key card is in the discard pile and you need it NOW. Retrieve Impervious for a big incoming attack, Bash+ to refresh Vulnerable, or a kill card for a lethal opportunity. The 0 cost makes it free value.

Do not plan around using Liquid Memories to recover exhausted cards (Intimidate, Shockwave, Impervious, Reaper) -- it cannot retrieve them from the exhaust pile. It will retrieve a different card from discard instead, wasting the potion.
