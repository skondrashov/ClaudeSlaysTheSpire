# Forgotten Altar

- If HP > 40 and a rest site is upcoming: Sacrifice. 20 HP loss is recoverable; +5 Max HP is permanent value.
- If HP <= 20: Desecrate (Sacrifice would kill you). Decay curse can be exhausted by [[cards/Fiend Fire]] or removed at shops.
- If HP 20-40 with no rest coming: Leave. Neither option is safe.
- If you have exhaust tools (Fiend Fire, [[cards/True Grit]], [[cards/Second Wind]]): Desecrate is less punishing since you can exhaust the curse in combat.

Priority (high HP): Sacrifice > Leave > Desecrate.
Priority (low HP, forced): Desecrate > Leave. Never Sacrifice below 21 HP.
