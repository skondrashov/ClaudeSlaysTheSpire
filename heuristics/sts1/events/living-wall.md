# Living Wall

If a game-defining upgrade exists ([[cards/True Grit]] -> True Grit+, [[cards/Shockwave]] -> Shockwave+), ALWAYS choose Grow. Otherwise, Forget (Remove) is the default -- deck thinning is permanent value. Change (Transform) is high variance and a gamble.

Priority: Grow (if critical upgrade target exists) > Forget (default) > Change (gambling).
