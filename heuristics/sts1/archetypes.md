# Winning Archetypes

Pursue whichever engine the card/relic offerings support. No single archetype is "the" Ironclad strategy.

## [[characters/Ironclad]]

**1. [[relics/Snecko Eye]] + [[buffs/Strength]] Scaling** — Snecko Eye (draw 7, cost randomization) + [[cards/Immolate]] (AOE) + [[cards/Limit Break]] (Str doubler) + [[cards/Heavy Blade]] finishers. Roll cheap costs on expensive cards, double Str for massive damage. Commit when: Snecko Eye is offered as Act 2 boss relic and Limit Break or Heavy Blade already exist.

**2. [[cards/Corruption]] + [[cards/Feel No Pain]] Engine** — Corruption (Skills free but exhaust) + Feel No Pain (block per exhaust) + optional [[cards/Barricade]] (block persists) + [[cards/Entrench]] (double block) + [[cards/Body Slam]] (damage = block). Builds massive block from free Skill exhausts. Known winning versions all included Str sources ([[cards/Spot Weakness]]/[[relics/Red Skull]]; [[relics/Brimstone]]/Spot Weakness). Whether this engine can win WITHOUT any Strength is under investigation -- see exploration directives. One variant uses Brimstone as the primary Str source, with the engine providing block generation. Commit when: Corruption + FNP appear by mid-[[acts/Act 2]].

**3. Pure [[buffs/Strength]] Burst** — [[cards/Spot Weakness]]+ stacking + [[cards/Heavy Blade]]+ finisher + strong defensive tools ([[cards/Impervious]], [[cards/Intimidate]]). No Snecko Eye, no Barricade, no Corruption. Wins by accumulating Str over turns, then delivering a single massive Heavy Blade hit. Viable when: multiple Str sources appear early and Heavy Blade is in the deck. [[potions/Duplication Potion]] + Heavy Blade+ at high Str is a one-shot kill combo (312 damage confirmed vs [[bosses/Awakened One]]).

## [[characters/Silent]]

**4. [[cards/Accuracy]] [[cards/Shiv]] Burst** — Accuracy x2 (+8 Shiv damage) + [[cards/After Image]] (innate) + [[cards/Wraith Form]] + [[cards/Phantasmal Killer]] + [[cards/Malaise]] + [[cards/Blade Dance]]. Shivs deal 12+ damage each, beats [[bosses/Time Eater]] because each Shiv is one card play dealing massive damage.

**5. [[relics/Dead Branch]] Exhaust Engine** — Dead Branch + exhaust cards + After Image + [[cards/Footwork]] x2 + [[cards/Well-Laid Plans]]. Dead Branch creates infinite card advantage. Specifically defeats [[bosses/Awakened One]]: dump Powers in Phase 1, zero Powers in Phase 2.

**6. [[cards/Echo Form]] Orb Engine** — Echo Form+ + [[cards/Seek]]/[[relics/Bottled Lightning]] + [[cards/Coolheaded]] + [[cards/Barrage]] + [[relics/Mummified Hand]] + Loop + [[relics/Data Disk]]. Echo Form doubles first card each turn. Seek+ guarantees finding key cards.

## [[characters/Watcher]]

Stance dance (Wrath/Calm cycling) is the primary engine. The engine assembles reliably and execution is clean. The Watcher is structurally harder to pilot than other characters: no built-in healing, Wrath doubles incoming damage on mistakes, and the stance-sequencing engine is uniquely vulnerable to hand/deck clog (Slimed, Burns, Dazed).

**Stance Dance Engine (unproven, very promising):** Eruption+ (1E Wrath entry) + Calm exits (Vigilance, Inner Peace, Empty Body) + Rushdown (draw on Wrath entry) + Mental Fortress (block on stance change) + Flurry of Blows (free damage on stance change) + Battle Hymn (sustained Smite generation) + Apparitions (Intangible for boss survival). The energy loop (Calm +2E -> Wrath doubled damage) is the Watcher's fundamental scaling path. Mental Fortress+ (6 block per stance change) is excellent as a passive defense layer.

**Key Powers (priority order):** Mental Fortress > Rushdown > Battle Hymn. Mental Fortress converts stance cycling into free block. Rushdown provides draw density for burst turns. Battle Hymn generates sustained boss damage.

## What All Wins Share

(1) Zero UI misplays, (2) zero arithmetic errors, (3) strong [[acts/Act 1]] performance, (4) proper map pathing, (5) a complete scaling engine -- not individual strong cards. Strength scaling is the most proven path. The Corruption+FNP engine can win but always alongside Str sources -- its independence is under investigation. Adding Brimstone as a Str source variant makes the engine + any Str source the most reliable Ironclad formula. The key is having SOME scaling path AND clean execution AND correct fight-specific knowledge.
