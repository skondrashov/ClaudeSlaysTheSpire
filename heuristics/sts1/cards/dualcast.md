# Dualcast

Burst damage (Lightning: 8x2 = 16) or burst block (Frost: 5x2 = 10) for 1E. Excellent finisher.

**When to play:** Only valuable when an orb is channeled. With an empty orb slot, Dualcast does nothing. Plan orb channeling before using -- channel Lightning with Zap (0E) then Dualcast for 16 damage. With Focus scaling, evoke values increase multiplicatively.

**Upgrade priority:** High. 1E->0E is massive -- free double evoke.

**Nob warning:** Dualcast is a Skill card. Against Nob, only play if the evoke damage justifies the +2 Str cost (e.g., killing Nob that turn).
