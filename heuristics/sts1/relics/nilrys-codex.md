# Nilry's Codex

Provides a free card every turn, excellent in long fights where hand quality matters. Generated cards include powerful options like [[cards/Impervious]] and [[cards/Immolate]] from any class pool. In a boss fight lasting 8+ turns: 8+ free cards, some game-changing.

High value in long fights (bosses, elites). The card generation compensates for thin decks or decks lacking specific answers. Lower value in short hallway fights (2-3 turns, only 2-3 extra cards).
