# Cursed Key

Strong when the deck can handle curse dilution. +1 Energy is transformative (4E to 5E enables [[cards/Barricade]]/[[cards/Corruption]] + block in one turn). The curse downside is manageable only if chests stay optional: curses can be removed at shops or via exhaust cards.

**Curses enter SILENTLY on chest opens — no prompt, no skip.** After any chest open with Cursed Key, deck-check for the new curse BEFORE buying shop removal; removal bought before the check gets spent on the wrong card while the curse rides to the boss.

**[[relics/Tiny Chest]] makes Cursed Key a liability.** Tiny Chest converts every 4th ? room into a forced Treasure room, turning "avoidable chest curses" into scheduled ones. Holding Tiny Chest (or planning Treasure nodes), price each chest as relic + curse before taking Cursed Key.

WHEN TO TAKE:
- Deck has exhaust tools ([[cards/Second Wind]], Corruption).
- Deck needs energy for 3E Powers (Barricade, Corruption, [[cards/Demon Form]]).
- Chest rooms are avoidable on the current map path — and no Tiny Chest.

NORMALITY CURSE WARNING: The Normality curse (limit 3 card plays/turn) is the most dangerous curse from Cursed Key. Cripples Barricade + Corruption engine by limiting skill exhaust chains. Remove at first shop opportunity.

**NORMALITY + CORRUPTION+FNP INTERACTION:** The Corruption+FNP engine depends on playing many free Skills per turn to generate block. Normality caps total card plays at 3 per turn, completely shutting down the engine. This interaction can cost 40 HP in a single Transient fight and end an otherwise winning run. With Cursed Key + Corruption engine: (1) avoid opening non-boss chests when possible, (2) if Normality enters the deck, exhaust it immediately when drawn ([[cards/Burning Pact]], [[cards/True Grit]], [[cards/Second Wind]]), (3) prioritize shop removal of Normality over all other curse removal. If the deck is Corruption+FNP and no exhaust tool for curses exists, reconsider whether Cursed Key is the right boss relic.

SYNERGY: [[relics/Omamori]] (negates 2 curses from chests), exhaust engines (Corruption makes curse removal trivial).
