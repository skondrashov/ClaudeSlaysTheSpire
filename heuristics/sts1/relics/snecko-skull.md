# Snecko Skull

Whenever you apply [[debuffs/Poison]], apply 1 additional. This is a generic trigger on the *application* event, so it stacks any poison source by +1 per application — Shivs/Bane do nothing, but every per-hit or per-card poison source benefits.

**[[cards/Envenom]] + Snecko Skull = 2 Poison per unblocked hit.** Envenom applies 1 Poison on each unblocked Attack hit; Snecko Skull adds 1 more per application, so each unblocked hit applies **2** Poison. With shiv volume (multi-hit Daggers, Blade Dance, Infinite Blades) this roughly **doubles the poison clock** — the difference between an 8+-turn poison engine and one that clears Act 2/3 elites on the 5-6-turn kill-speed budget.
