# Colosseum

After Wave 1, check HP:
1. Below 30 HP: ALWAYS escape (COWARDICE). The Wave 2 enemy pool includes [[enemies/Gremlin Nob]], which can kill any deck at low HP.
2. Check deck composition. Skill-heavy decks ([[characters/Defect]], early [[characters/Silent]]) face worst-case matchups in Wave 2. Escape unless HP is above 60%.
3. Check potions. If all defensive potions were used in Wave 1, escape -- Wave 2 requires fresh resources.
4. Fight only if HP is above 50% after Wave 1 and the deck can handle another multi-enemy fight.

The rewards from Wave 2 are NEVER worth a run-ending death. Colosseum appears mid-[[acts/Act 2]] where the remaining path has plenty of card/relic opportunities.

Warnings:
- Pre-boss HP drain: If encountered 1-2 floors before the boss, the forced combat can drain 15-25 HP. Unknown rooms carry Colosseum risk even with [[relics/Juzu Bracelet]].
- Entangle: [[enemies/Blue Slaver]] prevents all attacks for 1 turn. Plan for 1 turn of pure blocking.
- Gremlin Nob in Wave 2: Nob's [[buffs/Enrage]] (+2 Str per Skill) is devastating for Skill-heavy decks.
