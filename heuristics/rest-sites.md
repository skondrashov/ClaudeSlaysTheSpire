# Rest Site Decisions

- **Upgrade vs Rest Framework**: Upgrade when HP > 50% Act 2 (> 35% Act 1) and critical upgrade target exists. Rest when HP < 50% Act 2 (< 35% Act 1), or next room is elite/boss and below 60%, or path has 2+ combats before next rest and below 50%.
- **The Upgrade Death Spiral**: Zero upgrades = death sentence. Unupgraded cards → longer fights → more HP loss → forced resting → no upgrades. MANDATORY RULE: Upgrade at EVERY rest site above threshold. Bash should be upgraded by Floor 8.
- **Upgrade Priority (Ironclad)**: 1. Bash (3 Vuln vs 2). 2. Corruption (3E→2E). 3. True Grit (chosen vs random exhaust). 4. Armaments (1 card → ALL cards). 5. Carnage (28 vs 20). 6. Iron Wave (7/7 vs 5/5). 7. Pommel Strike (10+draw 2 vs 9+draw 1). 8. Thunderclap (7 vs 4 AOE). 9. SIO (11 vs 8). 10. Headbutt (12 vs 9). 11. Defend (8 vs 5). Context overrides: Corruption above Bash if it's the engine piece. Limit Break is #1 for Str-scaling decks approaching Act 3 boss.
