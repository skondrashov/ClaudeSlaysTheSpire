# Feel No Pain

Passive block engine that triggers on every exhaust. Permanent for the rest of combat.

**When to play:** On a free turn for zero opportunity cost. At 1E, worth playing even when only 2-3 exhaust triggers are expected. Priority increases with more exhaust sources.

**Synergies:** True Grit+ (exhaust + block + 3 FNP), Burning Pact (exhaust + draw + 3 block), Fiend Fire (exhaust entire hand = 3 per card), Corruption (all Skills exhaust = constant triggers -- FNP is critical for surviving Corruption), Corruption + Dead Branch (triple combo: free exhaust + 3 block + random card), Second Wind (mass exhaust = massive FNP burst).

**Slime Boss synergy:** Each Slimed exhausted generates 3 block, partially offsetting 1E tempo cost. 5+ Slimed = 15+ block across the fight.

**Spheric Guardian:** Corruption + FNP successfully defeated Spheric Guardian with only 12 HP lost. Recommended configuration for the hardest Act 2 hallway fight.

**Victory engine:** Mandatory component of Barricade + Corruption engine. FNP+ (4 block/exhaust) generated 40-60+ block per fight from exhaust triggers alone.

**Anti-synergy:** Weak in decks with few exhaust sources.
