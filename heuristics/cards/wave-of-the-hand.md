# Wave of the Hand

Play FIRST before any block cards. Converts block generation into Weak application for the rest of the turn.

**When to play:** FIRST, before any block-generating cards. It must be active before block is gained to trigger Weak. Playing it after block cards wastes the triggers.

**Upgrade priority:** Medium-high. 1->2 Weak per trigger doubles the Weak output. With 2-3 block sources per turn, this is 4-6 Weak instead of 2-3.

**Multiple triggers:** Every block source triggers a separate Weak application. Wave of the Hand + 2 Defends = 2 Weak applications. Wave + Vigilance + Halt = 3 applications.

**Just Lucky synergy:** Just Lucky gives 2-3 block, triggering Wave. Free Weak application from a 0E card.

**Hexaghost Inferno:** Play Wave before block cards on the Inferno turn. Each block card applies Weak, reducing Inferno's multi-hit damage significantly.

**Draft priority:** Medium-high. The Watcher's primary Weak source. Weak is critical for surviving Inferno and large attacks, so Wave fills a mandatory slot in boss preparation.
