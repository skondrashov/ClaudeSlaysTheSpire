# Talk to the Hand

Block Return per hit. Excellent against multi-hit enemies. Play early since it exhausts.

**When to play:** Early in the fight for maximum Block Return value over multiple attack turns. Since it exhausts, apply it to the primary threat immediately -- do not save for later.

**Upgrade priority:** Medium-high. Block Return 2->3 is 50% more free block per hit. Against multi-hit enemies, this scales rapidly.

**Snake Plant (7x3):** Block Return 2 triggers 3 times per Chomp = 6 free block per attack. Upgraded: 9 free block. Extremely strong in this matchup.

**Multi-hit enemies:** The more hits an enemy does, the more value Block Return provides. Excellent against Hexaghost Inferno (6 hits), Book of Stabbing, and any multi-attack enemy.

**Exhausts:** Only one use per combat. Apply to the enemy that attacks most frequently or with the most hits.
