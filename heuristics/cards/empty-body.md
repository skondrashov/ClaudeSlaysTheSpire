# Empty Body

Wrath exit that leaves you in no stance (neutral). Provides block without entering Calm.

**When to play:** When you need to exit Wrath safely but do not want the Calm energy cycle. Also useful as simple 0-cost-equivalent block (7/10 for 1E).

**Upgrade priority:** Low-medium. 7->10 block is a solid improvement but does not change the card's role.

**Play order (CRITICAL):** Play [[cards/Flurry of Blows]] BEFORE Empty Body. Leaving any stance triggers Flurry's return from discard.

**[[cards/Vigilance]] comparison:** Vigilance is generally better (enters Calm for +2E next turn). Empty Body provides flexibility when you do not want to be locked into the Calm->Wrath cycle.

**Draft priority:** Low-medium. Take only if Vigilance copies are insufficient for Wrath exits.
