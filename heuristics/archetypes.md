# Winning Archetypes

1. **[[characters/Ironclad]]: [[relics/Snecko Eye]] + [[buffs/Strength]] Scaling** — Snecko Eye (draw 7, cost randomization) + [[cards/Immolate]] (AOE) + [[cards/Limit Break]] (Str doubler) + [[cards/Heavy Blade]] finishers. Roll cheap costs on expensive cards, double Str for massive damage.
2. **Ironclad: [[cards/Barricade]] + [[cards/Corruption]] Engine** — Barricade (block persists) + Corruption (Skills free but exhaust) + [[cards/Feel No Pain]] (block per exhaust) + [[cards/Entrench]] (double block) + [[cards/Body Slam]] (damage = block). Builds 100-200+ block wall, Body Slam converts to damage.
3. **[[characters/Silent]]: [[cards/Accuracy]] [[cards/Shiv]] Burst** — Accuracy x2 (+8 Shiv damage) + [[cards/After Image]] (innate) + [[cards/Wraith Form]] + [[cards/Phantasmal Killer]] + [[cards/Malaise]] + [[cards/Blade Dance]]. Shivs deal 12+ damage each, beats [[bosses/Time Eater]] because each Shiv is one card play dealing massive damage.
4. **Silent: [[relics/Dead Branch]] Exhaust Engine** — Dead Branch + exhaust cards + After Image + [[cards/Footwork]] x2 + [[cards/Well-Laid Plans]]. Dead Branch creates infinite card advantage. Specifically defeats [[bosses/Awakened One]]: dump Powers in Phase 1, zero Powers in Phase 2.
5. **[[characters/Defect]]: [[cards/Echo Form]] Orb Engine** — Echo Form+ + [[cards/Seek]]/[[relics/Bottled Lightning]] + [[cards/Coolheaded]] + [[cards/Barrage]] + [[relics/Mummified Hand]] + Loop + [[relics/Data Disk]]. Echo Form doubles first card each turn. Seek+ guarantees finding key cards.

**What all formulas share:** (1) zero UI misplays, (2) zero arithmetic errors, (3) strong [[acts/Act 1]] performance, (4) proper map pathing, (5) a complete scaling engine (not individual strong cards). Pursue whichever engine the card/relic offerings support.
