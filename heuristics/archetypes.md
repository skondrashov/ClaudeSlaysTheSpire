# Winning Archetypes

1. **Ironclad: Snecko Eye + Strength Scaling** — Snecko Eye (draw 7, cost randomization) + Immolate+ (AOE) + Limit Break (Str doubler) + Heavy Blade finishers. Roll cheap costs on expensive cards, double Str for massive damage.
2. **Ironclad: Barricade + Corruption Engine** — Barricade (block persists) + Corruption (Skills free but exhaust) + Feel No Pain (block per exhaust) + Entrench (double block) + Body Slam+ (damage = block). Builds 100-200+ block wall, Body Slam converts to damage.
3. **Silent: Accuracy Shiv Burst** — Accuracy x2 (+8 Shiv damage) + After Image (innate) + Wraith Form+ + Phantasmal Killer + Malaise + Blade Dance. Shivs deal 12+ damage each, beats Time Eater because each Shiv is one card play dealing massive damage.
4. **Silent: Dead Branch Exhaust Engine** — Dead Branch + exhaust cards + After Image + Footwork x2 + Well-Laid Plans. Dead Branch creates infinite card advantage. Specifically defeats Awakened One: dump Powers in Phase 1, zero Powers in Phase 2.
5. **Defect: Echo Form+ Orb Engine** — Echo Form+ + Seek+/Bottled Lightning + Coolheaded+ + Barrage+ + Mummified Hand + Loop + Data Disk. Echo Form doubles first card each turn. Seek+ guarantees finding key cards.

**What all formulas share:** (1) zero UI misplays, (2) zero arithmetic errors, (3) strong Act 1 performance, (4) proper map pathing, (5) a complete scaling engine (not individual strong cards). Pursue whichever engine the card/relic offerings support.
