# Dead Branch

Transforms exhaust from a resource-spending mechanic into a resource-generating one. The more cards exhausted per fight, the more value generated.

**CORRUPTION SYNERGY (strongest combo in the game):** With Corruption, every Skill costs 0E and exhausts, and Dead Branch replaces it with a random card. Hand never empties, cycling enormous numbers of cards per turn. Any generated block/damage cards are effectively free. This engine wins any fight it survives to activate.

SETUP RISK: Corruption (3E unupgraded) + Dead Branch requires surviving the setup turn with no immediate block.

**FEEL NO PAIN SYNERGY:** Every exhaust triggers both Dead Branch (new card) AND Feel No Pain (3 block). Provides defensive sustain during engine turns.

**SILENT EXHAUST SYNERGY:** Shivs exhaust after play -- each triggers Dead Branch. Storm of Steel -> 5 Shivs -> 5 Dead Branch procs = massive card generation. Terror, Endless Agony, Adrenaline all provide repeatable triggers.

When you have Dead Branch, prioritize cards that exhaust (Corruption, Second Wind, True Grit+, Fiend Fire). Corruption becomes the highest-priority card to acquire.

Always a strong relic pick. Universally strong with no significant anti-synergies.
