# Fusion Hammer

Excellent when the deck is already well-upgraded. +1 Energy/turn is transformative. Rest sites become heal-only (and [[relics/Ancient Tea Set]] always triggers).

WHEN TO TAKE:
- Most key cards already upgraded ([[cards/Bash]], core block/damage cards).
- Deck needs energy more than upgrades.
- [[potions/Blessing of the Forge]] potion available (one-time mass upgrade substitute).
- Ancient Tea Set equipped (always triggers since you must rest).

WHEN TO AVOID:
- Critical cards still unupgraded (Bash, [[cards/Corruption]], [[cards/True Grit]], [[cards/Limit Break]]).
- Deck recently assembled and needs 3+ upgrades.

SYNERGIES:
- Ancient Tea Set (always triggers since you can only rest, not smith).
- [[cards/Armaments]] (upgrade ALL cards in hand, substitutes for smith in-combat).
- [[relics/Regal Pillow]] (extra healing since you always rest).
- **[[cards/Apotheosis]]** (becomes the single most valuable card -- provides the ONLY way to upgrade cards). Actively seek Apotheosis from colorless events when [[cards/Fusion]] Hammer is equipped.
