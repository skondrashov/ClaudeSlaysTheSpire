# Cursed Key

Strong when the deck can handle curse dilution. +1 Energy is transformative (4E to 5E enables [[cards/Barricade]]/[[cards/Corruption]] + block in one turn). The curse downside is manageable: only triggers on chests (optional), curses can be removed at shops or via exhaust cards.

WHEN TO TAKE:
- Deck has exhaust tools ([[cards/Second Wind]], Corruption).
- Deck needs energy for 3E Powers (Barricade, Corruption, [[cards/Demon Form]]).
- Chest rooms are avoidable on the current map path.

NORMALITY CURSE WARNING: The Normality curse (limit 3 card plays/turn) is the most dangerous curse from Cursed Key. Cripples Barricade + Corruption engine by limiting skill exhaust chains. Remove at first shop opportunity.

SYNERGY: [[relics/Omamori]] (negates 2 curses from chests), exhaust engines (Corruption makes curse removal trivial).
