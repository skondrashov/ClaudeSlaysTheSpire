# Draw Reduction

- **Effect:** Draw N fewer cards at the start of next turn. Removed after taking effect.
- **Duration:** 1 turn (applied this turn, takes effect next turn, then removed)
- **Sources (enemy):** Various enemies apply this as part of their attacks or abilities
- **Sources (player):** Some card downsides (e.g., [[cards/Scrape]] if conditions not met)
- **Stacking:** Additive. Multiple applications reduce draw further.
- **Interaction:** Only reduces the base draw (default 5). Cannot reduce draw below 0.
