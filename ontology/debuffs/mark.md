# Mark

- **Effect:** When Mark is applied to an enemy that already has Mark, that enemy takes damage equal to their current Mark stacks (before the new stacks are added).
- **Duration:** Permanent (does not decrement)
- **Stacking:** Additive. Each application adds stacks AND triggers damage equal to existing stacks.
- **Sources (player):** [[cards/Pressure Points]] card ([[characters/Watcher]] [[types/Skill]], 1E), applies 8/11 Mark
- **Interaction:** Scales quadratically — the more Mark already on an enemy, the more damage each new application deals. Bypasses [[buffs/Block]] (deals HP loss directly). Works across multiple enemies independently.
