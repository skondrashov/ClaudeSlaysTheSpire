# Dexterity

- **Effect:** Each point adds 1 [[buffs/Block]] to all block-gaining cards
- **Duration:** Permanent within combat, resets to 0 between combats
- **Negative Dexterity:** Reduces block from all block cards. Applied by [[enemies/Lagavulin]] debuff, [[debuffs/Wraith Form]] (-1/turn permanent).
- **Sources:** [[cards/Footwork]] (+2/+3), [[relics/Oddly Smooth Stone]] (+1 relic), [[relics/Data Disk]] (+1 relic, [[characters/Defect]])
