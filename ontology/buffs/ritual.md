# Ritual

- **Effect:** Gain N [[buffs/Strength]] at the start of each turn
- **Duration:** Permanent (does not decrement)
- **Stacking:** Additive. Ritual 3 + Ritual 2 = gain 5 [[buffs/Strength]] per turn.
- **Sources (enemy):** [[enemies/Cultist]] (gains Ritual 3 via Incantation)
- **Sources (player):** [[potions/Cultist Potion]] (gain Ritual 1)
