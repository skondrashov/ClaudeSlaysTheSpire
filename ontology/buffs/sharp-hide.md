# Sharp Hide

- **Effect:** Whenever the player plays an [[types/Attack]] card, the player takes N damage
- **Duration:** Active only during [[bosses/The Guardian]]'s Defensive Mode
- **Value:** 3 (A0)
- **Sources (enemy):** [[bosses/The Guardian]] (Defensive Mode only)
- **Interaction:** Triggers per [[types/Attack]] card played, not per hit. Multi-hit attacks trigger it once.
