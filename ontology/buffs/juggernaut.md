# Juggernaut

- **Effect:** Whenever you gain [[buffs/Block]], deal N damage to a random enemy.
- **Duration:** Permanent ([[types/Power]] effect, lasts rest of combat)
- **Sources (player):** [[cards/Juggernaut]] card ([[characters/Ironclad]] [[types/Power]], 2E), deals 5/7 damage per Block gain
- **Stacking:** Additive. Multiple instances increase damage per trigger.
- **Interaction:** Triggers on every source of Block gain — cards, [[buffs/Feel No Pain]], [[buffs/Metallicize]], etc. Each separate instance of Block gain triggers it once. Synergizes with [[buffs/Barricade]] builds that generate Block frequently.
