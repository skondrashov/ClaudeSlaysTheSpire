# Malleable

- **Effect:** Each time the entity is attacked within a single turn, gain increasing [[buffs/Block]] (first attack: N block, second: N+1, third: N+2, etc.). Resets at start of entity's turn.
- **Duration:** Permanent
- **Sources (enemy):** [[enemies/Writhing Mass]], [[enemies/Snake Plant]]
- **Implication:** Multiple small attacks in one turn give increasing block. Single large attacks per turn minimize block gained.
