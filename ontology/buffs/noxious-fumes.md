# Noxious Fumes

- **Effect:** At the start of each turn, apply N [[debuffs/Poison]] to ALL enemies.
- **Duration:** Permanent ([[types/Power]] effect, lasts rest of combat)
- **Sources (player):** [[cards/Noxious Fumes]] card ([[characters/Silent]] [[types/Power]], 1E), applies 2/3 [[debuffs/Poison]] per turn
- **Stacking:** Additive. Multiple Noxious Fumes increase Poison applied per turn.
- **Interaction:** Core scaling tool for Poison builds. Since [[debuffs/Poison]] decreases by 1 each turn, Noxious Fumes counters that decay and provides steady Poison growth. Synergizes with [[cards/Catalyst]] (multiply existing Poison) and [[debuffs/Corpse Explosion]].
