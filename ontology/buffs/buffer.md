# Buffer

- **Effect:** Prevents the next N instances of HP loss. Each time HP loss would occur, it is negated and Buffer decreases by 1.
- **Duration:** Permanent until all stacks consumed
- **Sources (player):** [[cards/Buffer]] card ([[characters/Defect]] [[types/Power]], 2E)
- **Sources (enemy):** Some enemies/bosses can gain Buffer
- **Interaction:** Prevents ALL HP loss including [[debuffs/Poison]] damage, not just attack damage. Does NOT prevent [[buffs/Block]] loss.
- **Stacking:** Additive. Multiple Buffer applications stack.
