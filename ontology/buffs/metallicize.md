# Metallicize

- **Effect:** Gain N [[buffs/Block]] at end of turn (after enemy intent resolves? No — at end of player's turn, before enemy attacks)
- **Duration:** Permanent ([[types/Power]] effect, lasts rest of combat)
- **Sources:** [[cards/Metallicize]] card (3/4 Block per turn), some enemies have innate Metallicize
- **Stacking:** Multiple sources stack additively. Two Metallicize = 6/8 Block per turn.
- **Interaction with [[buffs/Barricade]]:** Block from Metallicize persists between turns if Barricade is active.
- **NOT modified by [[buffs/Dexterity]]:** Metallicize is not a card play — it's a passive effect. Dexterity does not increase it.
- **NOT modified by [[debuffs/Frail]]:** Same reason. Frail only affects Block from cards.
