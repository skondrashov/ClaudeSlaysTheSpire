# Mode Shift

- **Effect:** A damage threshold. When the entity takes cumulative damage exceeding the threshold, it changes mode (attack mode ↔ defensive mode). The threshold resets (higher) each cycle.
- **Duration:** Permanent, cycles
- **Thresholds ([[bosses/The Guardian]]):** 30 (first), 40 (second), 50 (third)
- **Key mechanic:** If the threshold is triggered MID-ATTACK (a multi-hit attack pushes damage past the threshold), the current attack is cancelled.
- **Sources (enemy):** [[bosses/The Guardian]]
