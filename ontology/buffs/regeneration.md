# Regeneration

- **Effect:** At the end of each turn, heal N HP. Regeneration then decreases by 1. Removed when it reaches 0.
- **Duration:** Decreases by 1 each turn until 0
- **Sources (player):** [[cards/Reaper]] card heals based on damage dealt (different mechanic), [[cards/Self Repair]] ([[characters/Defect]]), [[potions/Regen Potion]]
- **Sources (enemy):** Various enemies/bosses can have Regeneration
- **Stacking:** Additive. Applying Regen 5 to existing Regen 3 = Regen 8.
- **Interaction:** Healing occurs at end of turn. Total healing from Regen N = N + (N-1) + ... + 1 = N*(N+1)/2.
