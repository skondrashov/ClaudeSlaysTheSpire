# Feel No Pain

- **Effect:** Whenever a card is Exhausted, gain N [[buffs/Block]].
- **Duration:** Permanent ([[types/Power]] effect, lasts rest of combat)
- **Sources (player):** [[cards/Feel No Pain]] card ([[characters/Ironclad]] [[types/Power]], 1E), grants 3/4 Block per Exhaust
- **Stacking:** Additive. Multiple instances stack the Block gained.
- **Interaction:** Core piece of Exhaust-based [[characters/Ironclad]] builds. Synergizes with [[cards/Corruption]] (all Skills exhaust = free Block), [[cards/Second Wind]], [[cards/Sentinel]], [[cards/True Grit]]. Combined with [[buffs/Barricade]], generates persistent Block rapidly.
