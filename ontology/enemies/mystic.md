# Mystic

- **Type:** Hallway
- **Act:** 2
- **HP:** 48-56

## Pattern

Heals if any enemy missing 16+ HP (priority). Otherwise: 60% Attack, 40% Buff. Cannot use Attack or Buff 3x in a row. Can Heal 2x in a row.

## Attacks

| Move | Damage | Effect |
|------|--------|--------|
| Attack | 8 | Apply 2 [[debuffs/Frail]] |
| Buff | -- | All enemies gain 2 [[buffs/Strength]] |
| Heal | -- | All enemies heal 16 HP |

## Mechanics

- Heal targets ALL enemies (including self)
- Heal threshold: any enemy missing 16+ HP
- Buff affects [[enemies/Centurion]] too — escalating damage if left alive
