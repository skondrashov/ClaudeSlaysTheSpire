# Giant Head

- **Type:** Elite
- **Act:** 3
- **HP:** 500

## Pattern

For the first 4 turns, uses Count (13 damage) or Glare (1 Weak) with 50/50 chance (cannot repeat either 3x in a row). After 4 turns, exclusively uses "It Is Time" for the rest of combat.
