# Block

Absorbs incoming damage before HP. Damage reduces Block first; excess hits HP.

## Player Block

- Resets to 0 at start of player's turn
- Modified by [[buffs/Dexterity]]: +1 per point to block gained from cards
- Modified by [[debuffs/Frail]]: x0.75 block from cards
- Persistence exceptions: [[buffs/Barricade]] (no reset), [[relics/Calipers]] (lose only 15), [[cards/Blur]] (no reset this turn)

## Enemy Block

- Resets to 0 at start of enemy's turn (same as player)
- Persistence exceptions: enemies with [[buffs/Barricade]] retain block between turns
