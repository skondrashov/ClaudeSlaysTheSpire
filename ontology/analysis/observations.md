# Observations

Working notes live in `analyst/observations.md`. Things noticed but not confirmed.

## Format

```
## Unconfirmed
- [item]: [what we think, what needs verification]

## Open Questions
- [question]: [context]
```

## Lifecycle

- New uncertain findings go in as Unconfirmed
- When a subsequent run confirms an observation, promote it to the appropriate heuristic file and remove it from observations
- Observations that are disproven get removed with a note about why
