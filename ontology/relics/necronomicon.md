# Necronomicon

- **Rarity:** Rare (Book relic)
- **Effect:** The first [[types/Attack]] card costing 2 or more energy played each turn is played a second time automatically. Adds a Necronomicurse (unplayable, cannot be removed except by special means) to the deck.
- **Obtained:** Cursed Tome event (costs 16 HP total)
- **Note:** Triggers only on the FIRST 2+ cost [[types/Attack]] each turn. The entire card effect replays (damage, draw effects, channel effects, debuffs).
