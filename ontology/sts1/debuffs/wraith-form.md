# Wraith Form

- **Effect:** Gain N stacks of [[buffs/Intangible]]. At the start of each subsequent turn, lose 1 [[buffs/Dexterity]] permanently.
- **Duration:** The [[buffs/Dexterity]] loss is permanent (stacks each turn for rest of combat)
- **Stacking:** Non-stacking
- **Interactions:** [[buffs/Artifact]] does NOT prevent the Dexterity loss (it is a self-applied effect from a [[types/Power]] you control, not an enemy debuff).
