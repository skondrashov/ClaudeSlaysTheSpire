# The Champ

- **Type:** Boss
- **Act:** 2
- **HP:** 420

## Attacks

| Move | Damage | Effect |
|------|--------|--------|
| Heavy Slash | 16 | -- |
| Face Slap | 12 | Applies 2 [[debuffs/Frail]] + 2 [[debuffs/Vulnerable]] |
| Defensive Stance | -- | Gains 15 Block + 5 [[buffs/Metallicize]] |
| Gloat | -- | Gains 2 [[buffs/Strength]] |
| Taunt | -- | Applies 2 [[debuffs/Weak]] + 2 [[debuffs/Vulnerable]] |
| Execute | 10x2 = 20 | -- |
| Anger | -- | Removes ALL debuffs, gains 6 [[buffs/Strength]] |

## Phase 1 (Above 50% HP)

Uses Taunt every 4 turns. On other turns: 15% Defensive Stance, 15% Gloat, 25% Face Slap, 45% Heavy Slash. No move used consecutively.

## Phase 2 -- Anger Transition (Below 50% HP)

When HP drops below 50%, performs Anger: removes ALL debuffs, gains 6 [[buffs/Strength]]. Then repeats: Random Move -> Random Move -> Execute (10x2 = 20 damage).
