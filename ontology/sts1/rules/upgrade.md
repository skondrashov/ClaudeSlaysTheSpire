# Upgrade

- **Effect:** A permanent improvement of a card to its enhanced form, marked with a "+" suffix (e.g. Bash → Bash+). The upgraded form replaces the base card in the deck.
- **Once per card:** A card can be upgraded only once. Exception: [[cards/Searing Blow]] can be upgraded any number of times, each adding damage.
- **Typical changes:** Most Attacks gain damage; most Skills gain Block or an improved effect; Powers gain magnitude. Some cards instead drop in cost, lose [[rules/Exhaust]], or gain [[rules/Retain]] / Innate.
- **Permanent vs. temporary:** A run upgrade lasts the whole run. Some sources upgrade cards only *for the rest of combat* — a temporary upgrade that reverts when the fight ends.
- **Sources:**
  - **Rest site (Smith):** upgrade one chosen card. The most common source.
  - **Cards (temporary, combat only):** [[cards/Armaments]] (a card in hand; upgraded, all cards in hand), [[cards/Apotheosis]] (all cards).
  - **Relics (on obtain):** [[relics/Whetstone]] (2 random Attacks), [[relics/War Paint]] (2 random Skills).
