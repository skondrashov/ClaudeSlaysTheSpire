# Red Slaver

- **Type:** Hallway
- **Act:** 1
- **HP:** 46-50

## Pattern

Always starts with Stab. Then repeats Scrape -> Scrape -> Stab until Entangle is used. Has 25% chance per turn to use Entangle instead of the normal pattern. After Entangle (used only once per combat): 55% Stab, 45% Scrape (cannot use same move 3x in a row).

## Attacks

| Move | Damage | Effect |
|------|--------|--------|
| Stab | 13 | -- |
| Scrape | 8 | Applies 1 [[debuffs/Vulnerable]] |
| Entangle | -- | Applies [[debuffs/Entangled]]. Once per combat. |
