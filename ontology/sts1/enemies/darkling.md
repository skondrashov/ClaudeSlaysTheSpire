# Darkling

- **Type:** Hallway
- **Act:** 3
- **HP:** 48-56

## Pattern

**Outer Darklings:** Turn 1: 50/50 Nip or Harden. Other turns: 30% Nip, 40% Chomp, 30% Harden. Cannot use Harden or Chomp consecutively. Nip cannot repeat 3x.
**Middle Darkling:** Same but cannot use Chomp.

## Attacks

| Move | Damage | Effect |
|------|--------|--------|
| Nip | 7-11 (random, set at combat start) | -- |
| Chomp | 8x2 | Outer only |
| Harden | -- | Gain 12 Block |
