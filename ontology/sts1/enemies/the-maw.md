# The Maw

- **Type:** Hallway
- **Act:** 3
- **HP:** 300 (does NOT change with Ascension)

## Pattern

- **Turn 1:** Roar (applies 3 [[debuffs/Weak]] + 3 [[debuffs/Frail]]). No damage.
- **Turns 2+:** After Drool -> 50/50 Slam or Nom Nom. After Slam -> 50/50 Nom Nom or Drool. After Nom Nom -> always Drool.
