# Mugger

- **Type:** Hallway
- **Act:** 2
- **HP:** 48-52

## Pattern

Turns 1-2: Mug (10 damage + steals gold). Turn 3+: 50% Lunge (16 damage) or 50% Smoke Bomb (11 Block). After Smoke Bomb, uses Escape next turn (flees with stolen gold). Same sequence as [[enemies/Looter]] but higher damage and block values.

## Attacks

| Move | Damage | Effect |
|------|--------|--------|
| Mug | 10 | Steals gold |
| Lunge | 16 | -- |
| Smoke Bomb | -- | Gains 11 Block |
| Escape | -- | Flees combat with stolen gold |

## Mechanics

Stolen gold is returned if killed. If it escapes, gold is lost permanently.
