# Flight

- **Effect:** ALL damage taken is halved (rounded down). Each time the entity takes a hit, Flight decreases by 1. When Flight reaches 0, damage is no longer halved.
- **Duration:** Reapplied each turn by the enemy
- **Stacking:** Non-stacking (reapplied to a fixed value each turn)
- **Interactions:** Each hit in a multi-hit attack decrements Flight separately and is halved individually.
