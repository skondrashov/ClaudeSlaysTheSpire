# Mode Shift

- **Effect:** A damage threshold. When the entity takes cumulative damage exceeding the threshold, it changes mode (attack mode to defensive mode or vice versa). The threshold resets (higher) each cycle.
- **Stacking:** Non-stacking
- **Thresholds ([[bosses/The Guardian]]):** 30 (first), 40 (second), 50 (third)
- **Interactions:** If the threshold is triggered mid-attack (a multi-hit attack pushes damage past the threshold), the remaining hits are cancelled.
