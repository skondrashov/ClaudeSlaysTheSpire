# Ontology Corrections

Ontology lives in `ontology/`. It is formally closed — entries describe what entities ARE and DO, using `[[category/Name]]` links. No opinions, no strategy.

## When to Change Ontology

Fix only factual errors:
- Wrong damage numbers
- Missing attacks or abilities
- Incorrect mechanics
- Wrong interactions

## When NOT to Change Ontology

Do NOT add strategic commentary. "Jaw Worm is dangerous" is a heuristic. "Jaw Worm attacks for 11" is ontology.
