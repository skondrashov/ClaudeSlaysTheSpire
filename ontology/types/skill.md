# Skill

A card type. Skills typically provide block, apply debuffs, or generate utility effects.

## Mechanical interactions

- Block from Skills is modified by Dexterity (→ effects/dexterity.md): +1 block per point
- Block from Skills is modified by Frail (→ effects/frail.md) on the player: x0.75
- Triggers Gremlin Nob's Enrage (+2 Strength to Nob per Skill played)
- Triggers Hex (Chosen): adds a Dazed to draw pile per Skill played
- Does NOT benefit from Strength
- Does NOT interact with Vulnerable
- Corruption (Ironclad Power) makes all Skills cost 0 but Exhaust after playing
